# Budget Template

## Columns

| Column        | Required | Notes                                             |
| ------------- | -------- | ------------------------------------------------- |
| Period        | Yes      | e.g. `2026-W01` or `2026-01`                      |
| Category      | Yes      | revenue, payroll, expenses, capex, loan_repayment |
| Account Code  | Optional | Link to COA                                       |
| Budget Amount | Yes      | Numeric                                           |
| Budget Type   | Yes      | e.g. operating, capex                             |

## Usage

Feeds budget vs actual engine after trial balance actuals imported.
